public function addDCElements(datasetName:String, elements:Object, callback:Function = null):void

Adds the given Dublin Core key-value pairs to the metadata store for the dataset with the given name.

Parameters
datasetName:String — the name of the dataset to associate the new element values with
 
elements:Object — an Object (map) whose keys are strings such as "dc:title", "dc:description", etc. and whose values are the (String) values for those elements, applied to the dataset with the given name.
 
callback:Function (default = null) — (optional) a function called when the server returns (of signature function(e:Event, token:Object = null):void)
