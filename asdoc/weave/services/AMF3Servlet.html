Method detail
invokeAsyncMethod()method
public override function invokeAsyncMethod(methodName:String, methodParameters:Object = null):AsyncToken

This function makes a remote procedure call to a Weave AMF3 servlet. As a special case, if methodParameters is an Array and the last item is a ByteArray, the bytes will be appended after the initial AMF3 serialization of the preceeding parameters to allow additional content that can be treated as a stream in Java.

Parameters
methodName:String — The name of the method to call.
 
methodParameters:Object (default = null) — The parameters to use when calling the method.

Returns
AsyncToken — An AsyncToken generated for the call.
