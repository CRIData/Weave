Packageweave.services.wms
Classpublic class WMSTileIndex

This class provides an interface to a collection of WMSTiles. The tiles are inserted into the 5-dimensional KDTree with keys of the form [xMin, yMin, xMax, yMax, zoomLevel].

getTilesWithinBounds()method 
public function getTilesWithinBounds(dataBounds:IBounds2D, zoomLevel:Number):Array

Parameters
dataBounds:IBounds2D — The bounds of the data.
 
zoomLevel:Number — The zoom level of the data. This parameter should be the same for all tiles located at one zoom level. For NASA WMS, this is preferably defined as the area of the best matching bbox divided by the area of the image. For the ModestMaps services, this should be the constant zoom level provided in a Coordinate object.

Returns
Array — An array of keys with bounds that contain the given bounds.
