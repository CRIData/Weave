getValueFromKey()method 
public override function getValueFromKey(key:IQualifiedKey, dataType:Class = null):*

This function returns different results depending on the dataType. Supported types: default -> IBinClassifier that matches the given record key Number -> bin index for the given record key String -> bin name for the given record key Array -> list of keys in the same bin as the given record key

Parameters
key:IQualifiedKey — A record identifier.
 
dataType:Class (default = null) — The requested return type.

Returns
* — If the specified dataType is supported, a value of that type. Otherwise, the default return value for the given record key.
