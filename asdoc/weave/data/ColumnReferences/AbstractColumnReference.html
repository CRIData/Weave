Packageweave.data.ColumnReferences
Classpublic class AbstractColumnReference
ImplementsIColumnReference
SubclassesHierarchyColumnReference

This is a base class for implementing an IColumnReference. This class should be extended and include all the information required to retrieve a column of data. The implementation of getHashCode() uses the session state of this object to generate a hash code. When you extend AbstractColumnReference, you need to make sure that the session state contains all the properties required to differentiate one column reference from another.
