Packageweave.data.DataSources
Classpublic class AbstractDataSource
ImplementsIDataSource, IDisposableObject
SubclassesCSVDataSource, DBFDataSource, GrailsDataSource, WeaveDataSource, WFSDataSource, XLSDataSource

This is a base class to make it easier to develope a new class that implements IDataSource. To extend this class, the minimum functions to override are: initialize(), requestHierarchyFromSource() and requestColumnFromSource(). Optionally, initializationComplete() can also be overridden to control how long column requests are delayed.

initializationCompleteproperty 
initializationComplete:Boolean  [read-only]

Classes that extend AbstractDataSource can define their own replacement for this function. All column requests will be delayed as long as this accessor function returns false. The default behavior is to return false during the time between a change in the session state and when initialize() is called.
