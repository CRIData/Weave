requestColumnFromSource()method 
protected override function requestColumnFromSource(columnReference:IColumnReference, proxyColumn:ProxyColumn):void

requestColumnFromSource This function must be implemented by classes by extend AbstractDataSource. This function should make a request to the source to fill in the proxy column.

Parameters
columnReference:IColumnReference — An object that contains all the information required to request the column from this IDataSource.
 
proxyColumn:ProxyColumn — ProxyColumn object that will be updated when the column data is ready.

Parameters
