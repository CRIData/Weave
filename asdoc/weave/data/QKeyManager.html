Packageweave.data
Classpublic final class QKeyManager
ImplementsIQualifiedKeyManager

This class manages a global list of IQualifiedKey objects. The getQKey() function must be used to get IQualifiedKey objects. Each QKey returned by getQKey() with the same parameters will be the same object, so IQualifiedKeys can be compared with the == operator or used as keys in a Dictionary.
