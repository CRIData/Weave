Property detail
autoBalanceproperty
public var autoBalance:Boolean = false

If this is true, the tree will automatically balance itself when queried after nodes are inserted. NOTE: Balancing a large tree is very slow, so this will not give any benefit if the tree changes often.

Method detail
balance()method
public function balance():void

Balance the tree so there are an (approximately) equal number of points on either side of any given node. A balanced tree yields faster query times compared to an unbalanced tree. NOTE: Balancing a large tree is very slow, so this should not be called very often.

public function insert(key:Array, obj:Object):KDNode
