public function reportError(error:Object, faultMessage:String = null, faultContent:Object = null):void

This function is intended to be the global error reporting mechanism for Weave.

Parameters
error:Object — An Error or a String describing the error.
 
faultMessage:String (default = null) — A message associated with the error, if any. If specified, the error will be wrapped in a Fault object.
 
faultContent:Object (default = null) — Content associated with the error, if any. If specified, the error will be wrapped in a Fault object.
