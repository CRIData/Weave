Packageweave.api.primitives
Interfacepublic interface IMatrix
ImplementorsWeaveMatrix

IMatrix provides an interface to a two dimensional array of numbers. Operations allowed on an IMatrix are addition, subtraction, multiplication, and inversion. All of these operations are in-place on this matrix and return this matrix to allow nesting. Uninitialized entries in the array default to 0 to imitate sparse matrices.
