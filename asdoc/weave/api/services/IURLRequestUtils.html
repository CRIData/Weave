getContent()method
public function getContent(request:URLRequest, asyncResultHandler:Function = null, asyncFaultHandler:Function = null, token:Object = null, useCache:Boolean = true, reportProgress:Boolean = true):IURLRequestToken

This function will download content from a URL and call the given handler functions when it completes or a fault occurrs.

Parameters
request:URLRequest — The URL from which to get content.
 
asyncResultHandler:Function (default = null) — A function with the following signature: function(e:ResultEvent, token:Object = null):void. This function will be called if the request succeeds.
 
asyncFaultHandler:Function (default = null) — A function with the following signature: function(e:FaultEvent, token:Object = null):void. This function will be called if there is an error.
 
token:Object (default = null) — An object that gets passed to the handler functions.
 
useCache:Boolean (default = true) — A boolean indicating whether to use the cached images. If set to true, this function will return null if there is already a bitmap for the request.
 
reportProgress:Boolean (default = true) — If set to true, WeaveAPI.ProgressIndicator will be notified of the download progress.

Returns
IURLRequestToken — An IURLRequestToken that can be used to cancel the request and cancel the async handlers.
getURL()method 
public function getURL(request:URLRequest, asyncResultHandler:Function = null, asyncFaultHandler:Function = null, token:Object = null, dataFormat:String = "binary", reportProgress:Boolean = true):URLLoader

This function performs an HTTP GET request and calls result or fault handlers when the request succeeds or fails.

Parameters
request:URLRequest — The URL to get.
 
asyncResultHandler:Function (default = null) — A function with the following signature: function(e:ResultEvent, token:Object = null):void. This function will be called if the request succeeds.
 
asyncFaultHandler:Function (default = null) — A function with the following signature: function(e:FaultEvent, token:Object = null):void. This function will be called if there is an error.
 
token:Object (default = null) — An object that gets passed to the handler functions.
 
dataFormat:String (default = "binary") — The value to set as the dataFormat property of a URLLoader object.
 
reportProgress:Boolean (default = true) — If set to true, WeaveAPI.ProgressIndicator will be notified of the download progress.

Returns
URLLoader — The URLLoader used to perform the HTTP GET request.
