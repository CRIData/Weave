transformBounds()method 
public function transformBounds(sourceSRS:String, destinationSRS:String, inputAndOutput:IBounds2D, xGridSize:int = 32, yGridSize:int = 32):IBounds2D

This function will approximately transform bounds from the sourceSRS to the destinationSRS.

Parameters
sourceSRS:String — The SRS code of the source projection.
 
destinationSRS:String — The SRS code of the destination projection.
 
inputAndOutput:IBounds2D — The bounds to transform. This is then used as the return value.
 
xGridSize:int (default = 32) — The x dimension of the grid used to approximate the transform.
 
yGridSize:int (default = 32) — The y dimension of the grid used to approximate the transform.

Returns
IBounds2D — The transformed bounds, inputAndOutput.
