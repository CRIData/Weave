Packageweave.api.data
Interfacepublic interface IProgressIndicator extends ILinkableObject
ImplementorsProgressIndicator

This is an interface for a central location to report progress of asynchronous requests. Since this interface extends ILinkableObject, getCallbackCollection() can be used on an IProgressIndicator.
