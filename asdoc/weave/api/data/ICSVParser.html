convertRecordsToRows()method
public function convertRecordsToRows(records:Array, columnOrder:Array = null, allowBlankColumns:Boolean = false):Array

This function converts an Array of Objects (compatible with DataGrid) to an Array of Arrays compatible with other functions in this class.

Parameters
records:Array — An Array of Objects containing String properties.
 
columnOrder:Array (default = null) — An optional list of column names to use in order.
 
allowBlankColumns:Boolean (default = false) — If this is set to true, then the function will include all columns even if they are blank.

Returns
Array — An Array of Arrays, the first being a header line containing all the property names.
