getAttributeColumn()method
public function getAttributeColumn(columnReference:IColumnReference):IAttributeColumn

The parameter to this function used to be pathInHierarchy because old implementations use XML path objects. The parameter type is now temporarily Object during this transitional phase. In future versions, the parameter will be an IColumnReference object.

Parameters
columnReference:IColumnReference — A reference to a column in this IDataSource.

Returns
IAttributeColumn — An IAttributeColumn object that will be updated when the column data downloads.
