Interfacepublic interface IPlotLayer extends mx.core.IUIComponent, ILinkableObject, IZoomView
ImplementorsPlotLayer, SelectablePlotLayer

A PlotLayer is a UIComponent that draws the graphics of an IPlotter. NOTE: This is a temporary interface just to get everything working. This interface should be redesigned to avoid dependencies on implementations such as SpatialIndex.
