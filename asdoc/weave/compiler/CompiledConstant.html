Packageweave.compiler
Classpublic class CompiledConstant
ImplementsICompiledObject

This serves as a wrapper for a constant value to remember that it was compiled. This is used in the Compiler class to avoid parsing tokens multiple times. To avoid function call overhead, no public functions are defined in this class.
