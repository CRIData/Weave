public function compileObjectToFunction(compiledObject:ICompiledObject, symbolTable:Object, ignoreRuntimeErrors:Boolean):Function

Parameters

Returns
Function — A Function that takes any number of parameters and returns the result of evaluating the ICompiledObject.
compileToFunction()method 
public function compileToFunction(expression:String, symbolTable:Object, ignoreRuntimeErrors:Boolean):Function

This function compiles an expression into a Function that evaluates using variables from a symbolTable. Strings may be surrounded by quotation marks (") and literal quotation marks are escaped by two quote marks together (""). The escape sequence for a quoted variable name to indicate a quotation mark is two quotation marks together.

Parameters
expression:String — An expression to compile.
 
symbolTable:Object — This is either a function that returns a variable by name or a lookup table containing custom variables and functions that can be used in the expression. These values may be changed after compiling.
 
ignoreRuntimeErrors:Boolean — If this is set to true, the generated function will ignore any Errors caused by the individual function calls in its execution. Return values from failed function calls will be treated as undefined.

Returns
Function — A Function generated from the expression String, or null if the String does not represent a valid expression.
compileToObject()method 
public function compileToObject(expression:String):ICompiledObject

This function will compile an expression into a compiled object representing a function that takes no parameters and returns a value. This function is useful for inspecting the structure of the compiled function and decompiling individual parts.

Parameters
expression:String — An expression to parse.

Returns
ICompiledObject — A CompiledConstant or CompiledFunctionCall generated from the tokens, or null if the tokens do not represent a valid expression.
