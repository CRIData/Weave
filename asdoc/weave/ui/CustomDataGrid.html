Classpublic class CustomDataGrid

This is a wrapper around a DataGrid to fix a bug with the mx_internal addMask() function which was introduced in Flex 3.6 SDK. The issue is the lockedColumnContent is instantiated and contains invalid data when the lockedColumnCount is 0.
