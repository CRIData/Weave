Packageweave.core
Classpublic class StageUtils

This is an all-static class that allows you to add callbacks that will be called when an event occurs on the stage. WARNING: These callbacks will trigger on every mouse and keyboard event that occurs on the stage. Developers should not add any callbacks that run computationally expensive code.

Method detail
addEventCallback()method
public static function addEventCallback(eventType:String, relevantContext:Object, callback:Function, parameters:Array = null, runCallbackNow:Boolean = false):void

WARNING: These callbacks will trigger on every mouse event that occurs on the stage. Developers should not add any callbacks that run computationally expensive code. This function will add a callback using the given function and parameters. Any callback previously added for the same function will be overwritten.

Parameters
eventType:String — The name of the event to add a callback for, one of the static values in the MouseEvent class.
 
relevantContext:Object — The function to call when an event of the specified type is dispatched from the stage.
 
callback:Function — An array of parameters that will be used as parameters to the callback function.
 
parameters:Array (default = null) — If this is set to true, the callback will be run immediately after it is added.
 
runCallbackNow:Boolean (default = false)
callLater()method 
public static function callLater(relevantContext:Object, method:Function, parameters:Array = null, allowMultipleFrameDelay:Boolean = true):void

This calls a function in a future ENTER_FRAME event. The function call will be delayed further frames if the maxComputationTimePerFrame time limit is reached in a given frame.

Parameters
relevantContext:Object — This parameter may be null. If the relevantContext object gets disposed of, the specified method will not be called.
 
method:Function — The function to call later.
 
parameters:Array (default = null) — The parameters to pass to the function.
 
allowMultipleFrameDelay:Boolean (default = true)

startTask()method 
public static function startTask(relevantContext:Object, task:IAsyncTask):void

This will start an asynchronous task, calling task.iterate() across multiple frames until it returns a value of 1 or the relevantContext object is disposed of.

Parameters
relevantContext:Object — This parameter may be null. If the relevantContext object gets disposed of, the task will no longer be iterated.
 
task:IAsyncTask — The asynchronous task.
Constant detail
maxComputationTimePerFrameconstant
public static const maxComputationTimePerFrame:int = 100

This is the recommended upper bound of computation time per frame. The "get shouldCallLater()" function uses this value along with currentFrameElapsedTime to determine its recommendation.

POINT_CLICK_EVENTconstant 
public static const POINT_CLICK_EVENT:String = "pointClick"

This is a special pseudo-event supported by StageUtils. Callbacks added to this event will only trigger when the mouse was clicked and released at the same screen location.
