Classpublic class LinkableEventListener
InheritanceLinkableEventListener CallbackCollection
ImplementsIDisposableObject, ILinkableObject

A LinkableEventListener is a sessioned eventLinster which is added to the stage. The action is specified by event.value and the listener function specified by the LinkableFunction macro.value.
