Classpublic class GridForceDirectedLayout
InheritanceGridForceDirectedLayout AbstractGraphAlgorithm CallbackCollection
ImplementsIGraphAlgorithm

This class is a force directed layout algorithm for graphs. The algorithm is the one suggested by Fructherman and Reingold for large graphs by using a grid to compute only certain repulsive forces. This algorithm is almost as aesthetically pleasing as ForceDirectedLayout, but it loses some accuracy and doesn't handle isolated nodes well.
