arcTo()method
public static function arcTo(graphics:Graphics, continueLine:Boolean, xCenter:Number, yCenter:Number, startAngle:Number, endAngle:Number, radius:Number, yRadius:Number):void

Similar to lineTo() and curveTo(), this will draw an arc on a Graphics object.

Parameters
graphics:Graphics — The Graphics where the arc will be drawn
 
continueLine:Boolean — If this is true, lineTo() will be used on the first coordinate instead of moveTo()
 
xCenter:Number — The x center coord of the arc
 
yCenter:Number — The y center coord of the arc
 
startAngle:Number — The angle where the arc starts
 
endAngle:Number — The angle where the arc ends
 
radius:Number — The radius of the circle that contains the arc
 
yRadius:Number — Optional y radius for an elliptical arc instead of a circular one
