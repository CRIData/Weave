Classpublic class SpatialIndex
InheritanceSpatialIndex CallbackCollection
ImplementsISpatialIndex

This class provides an interface to a collection of spatially indexed IShape objects. This class will not detect changes to the shapes you add to the index. If you change the bounds of the shapes, you will need to call SpatialIndex.createIndex().

getClosestOverlappingKeys()method 
public function getClosestOverlappingKeys(bounds:IBounds2D, xPrecision:Number, yPrecision:Number):Array

This function will get the keys closest the center of the bounds object. Generally this function will return an array of at most one key. Sometimes, it may return more than one key if there are multiple keys with equivalent distance to the center of the bounds object.

Parameters
bounds:IBounds2D — A bounds used to query the spatial index.
 
xPrecision:Number — If specified, X distance values will be divided by this and truncated before comparing.
 
yPrecision:Number — If specified, Y distance values will be divided by this and truncated before comparing.

Returns
Array — An array of IQualifiedKey objects.
