This function will determine whether the two lines intersect. An intersection is defined if the line segments cross at one point--if they are parallel or coincidental, the intersection is defined to be false.

lineIntersectsLine2()method 
public static function lineIntersectsLine2(Ax:Number, Ay:Number, Bx:Number, By:Number, Ex:Number, Ey:Number, Fx:Number, Fy:Number, outputIntersectionPoint:Point = null, ABasSeg:Boolean = true, EFasSeg:Boolean = true):Point

lineIntersectsLine This function computes the intersection point of two lines. Each line can individually be treated as a segment or an infinite line. It is much faster to compute the intersection of infinite lines. The algorithm was taken from the following blogs and optimized for speed: http://keith-hair.net/blog/2008/08/04/find-intersection-point-of-two-lines-in-as3/ http://blog.controul.com/2009/05/line-segment-intersection/

Parameters
Ax:Number — X coordinate of A in line AB
 
Ay:Number — Y coordinate of A in line AB
 
Bx:Number — X coordinate of B in line AB
 
By:Number — Y coordinate of B in line AB
 
Ex:Number — X coordinate of E in line EF
 
Ey:Number — Y coordinate of E in line EF
 
Fx:Number — X coordinate of F in line EF
 
Fy:Number — Y coordinate of F in line EF
 
outputIntersectionPoint:Point (default = null) — A Point object to store the intersection coordinates in
 
ABasSeg:Boolean (default = true) — true if you want to treat line AB as a segment, false for an infinite line
 
EFasSeg:Boolean (default = true) — true if you want to treat line EF as a segment, false for an infinite line

Returns
Point — Either outputIntersectionPoint or a new Point containing the intersection coordinates
