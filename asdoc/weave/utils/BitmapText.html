public function getUnrotatedBounds(outputBounds:IBounds2D):void

This function retrieves unrotated min,max coordinates corresponding to the text that is currently saved in the BitmapText object. Rotation is not considered when generating these coordinates.

Parameters
outputBounds:IBounds2D — A Bounds2D object to store the resulting coordinates in.
