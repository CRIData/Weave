drawCenteredIcon()method
public static function drawCenteredIcon(graphics:Graphics, x:Number, y:Number, source:BitmapData, fillColor:Number, fillAlpha:Number = 1.0, scale:Number = 1.0):void

This function will draw an image centered on an x,y coordinate.

Parameters
graphics:Graphics — The Graphics object where the icon should be drawn.
 
x:Number — The center x coordinate used to draw the icon.
 
y:Number — The center y coordinate used to draw the icon.
 
source:BitmapData — The icon image.
 
fillColor:Number — A background fill color.
 
fillAlpha:Number (default = 1.0) — A background fill alpha.
 
scale:Number (default = 1.0) — A scale value used to resize the source image.
getBitmapDataFromComponent()method 
public static function getBitmapDataFromComponent(component:UIComponent, desiredWidth:int = 0, desiredHeight:int = 0):BitmapData

This function gets a screenshot of a UIComponent and optionally creates a thumbnail version.

Parameters
component:UIComponent — A component from which to get a screenshot.
 
desiredWidth:int (default = 0) — If > 0, the width of the thumbnail version.
 
desiredHeight:int (default = 0) — If > 0, the height of the thumbnail version.

Returns
BitmapData — A screenshot of the component. If desired width or height is specified, a thumbnail version is returned.
resizeBitmapData()method 
public static function resizeBitmapData(source:BitmapData, width:uint, height:uint):BitmapData

This function will create a high quality thumbnail for a BitmapData.

Parameters
source:BitmapData — A BitmapData to create a thumbnail for.
 
width:uint — The maximum desired width of the result.
 
height:uint — The maximum desired height of the result.

Returns
BitmapData — A thumbnail version of the source BitmapData that fits within the specified width and height.
