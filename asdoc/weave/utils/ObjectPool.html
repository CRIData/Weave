Packageweave.utils
Classpublic class ObjectPool

This class contains static functions for pooling reusable objects. The purpose of this class is to avoid unnecessary garbage-collection activity that slows performance. It is recommended that you only use this class for simple, reusable objects that do not contain pointers to other objects. Use this class with care to prevent memory leaks.

returnObject()method 
public static function returnObject(object:Object):void

This function tells the ObjectPool that an object is no longer in use and it can go back into the pool so some other code can use it. This function should only be used on objects that you are sure no other code is still using. Also, any object you pass in here should not contain any pointers to other objects. If your code never uses borrowObject(), there is no reason to call returnObject(). If returnObject() is called more often than borrowObject(), that is basically a memory leak.

Parameters
object:Object — A simple reusable object that is no longer in use.
